# Privacy Policy for One More Sheep

Effective date: April 24, 2026

One More Sheep is a simple offline game. This application does not require account registration and does not collect, store, transmit, or share personal data with the developer or third parties.

## Data Collection

One More Sheep does not collect personal data such as:
- name
- email address
- phone number
- location
- contacts
- photos
- messages
- payment information

The app does not use analytics, advertising SDKs, external servers, or tracking tools.

## Local Storage

The app stores only a local high score on the user's device so the game can remember the best result. This information stays on the device and is not transmitted anywhere.

## Data Sharing

One More Sheep does not share any user data with third parties.

## Children

One More Sheep does not knowingly collect personal information from children or any other users.

## Security

Since the app does not collect or transmit personal data, no personal data is processed on external servers.

If this Privacy Policy changes in the future, the updated version will be published at this URL.
